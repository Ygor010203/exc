# Super Contador

Projeto desenvolvido durante o curso de JavaScript.

## Funcionalidades
- Contagem personalizada
- Definição de início, fim e passo
- Validação de campos vazios
- Ajuste automático quando o passo é inválido

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Como executar
1. Abra o arquivo `index.html`
2. Informe início, fim e passo
3. Clique em **Contar**
